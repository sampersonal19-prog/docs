---
title: "GPT Image 2 Text-to-Image"
api: "POST https://api.kenerateai.com/api/v2/openai/gpt-image-2/text-to-image"
description: "Generate highly detailed, photorealistic images from natural language text descriptions."
---

This endpoint leverages the GPT Image 2 model, featuring an **O-series planning stage** for perfect layouts and **99% text rendering accuracy** (multilingual). It is highly optimized for production environments with no cold start.

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer authentication header. Example: `Bearer YOUR_API_KEY`
</ParamField>

<ParamField header="Content-Type" type="string" required>
  Must be `application/json`
</ParamField>

### Body

<ParamField body="prompt" type="string" required>
  The text description of the desired image. To render typography or signs, enclose the text in quotes within your prompt.
</ParamField>

<ParamField body="aspect_ratio" type="string" default="1:1">
  The aspect ratio of the generated image. Supported values include `1:1`, `16:9`, `9:16`, `3:4`, `4:3`, `3:2`, `2:3`, `21:9`. Flexible from `3:1` to `1:3`.
</ParamField>

<ParamField body="resolution" type="string" default="1k">
  The resolution of the generated image. Supported values: `1k`, `2k`, `4k`.
</ParamField>

<ParamField body="quality" type="string" default="medium">
  The quality/steps of the generation. Supported values: `low`, `medium`, `high`. Higher quality uses more diffusion steps resulting in more detail and better lighting, but takes longer and costs more.
</ParamField>

<ParamField body="output_format" type="string" default="png">
  The file format of the output image. Supported values: `png`, `jpeg`, `webp`.
</ParamField>

<ParamField body="webhook_url" type="string">
  A URL that KenerateAI will send an HTTP POST request to when the generation is complete.
</ParamField>

<ParamField body="enable_base64_output" type="boolean" default="false">
  If true, the webhook payload or synchronous response will contain a raw Base64 data URI string instead of a public CDN URL.
</ParamField>

<ParamField body="enable_sync_mode" type="boolean" default="false">
  If true, the connection will remain open until the image generation is completely finished. **Note:** Only recommended for low-quality/low-resolution requests due to standard HTTP timeout limits (e.g. 60 seconds). Webhooks are preferred.
</ParamField>

### Request Example

```bash cURL
curl --request POST \
  --url https://api.kenerateai.com/api/v2/openai/gpt-image-2/text-to-image \
  --header 'Authorization: Bearer YOUR_API_KEY' \
  --header 'Content-Type: application/json' \
  --data '{
    "prompt": "A futuristic city skyline at neon sunset, highly detailed, photorealistic",
    "aspect_ratio": "16:9",
    "resolution": "2k",
    "quality": "high",
    "webhook_url": "https://your.app/webhooks/kenerate"
  }'
```

### Response

<ResponseField name="id" type="string">
  The unique identifier for the task. Use this to poll for status.
</ResponseField>

<ResponseField name="status" type="string">
  The current status of the task. Usually `queued` or `processing`.
</ResponseField>

```json Example Response
{
  "id": "task_1234567890",
  "status": "queued"
}
```
