## Overview

Bring your photos to life with smooth motion, natural transitions, and high visual fidelity — all guided by simple text prompts.

### Why It Stands Out
- **Image-driven generation**: Start from any image and animate it into a coherent video while preserving the original style and composition.
- **Prompt-guided motion**: Describe the action you want — camera movements, character expressions, environmental changes — and watch your vision unfold.
- **Negative prompt support**: Exclude unwanted elements or artifacts for cleaner, more controlled outputs.
- **Prompt Enhancer**: Built-in AI-powered prompt optimization helps you craft better descriptions for improved results.
- **Flexible duration**: Generate 5-second or 10-second clips depending on your needs.
- **Reproducibility**: Use the seed parameter to recreate exact results or iterate on variations.

## Quick Start

### API Endpoint
```http
POST https://api.kenerateai.com/api/v3/kenerateai/wan-2.1/i2v-720p
```

### Authentication
Include your API key in the `Authorization` header:
```http
Authorization: Bearer YOUR_API_KEY
```

## Request Schema

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `image` | string | **Yes** | Source image (public URL). |
| `prompt` | string | **Yes** | Text description of desired motion and style. |
| `negative_prompt` | string | No | Elements to avoid in the generated video. |
| `size` | string | No | Output resolution (default: 1280*720). |
| `num_inference_steps` | integer | No | Quality/speed trade-off (default: 30). |
| `duration` | string | No | Video length in seconds: 5 or 10 (default: 5). |
| `guidance_scale` | float | No | Prompt adherence strength (default: 5.0). |
| `flow_shift` | float | No | Motion intensity control (default: 5.0). |
| `seed` | integer | No | Set for reproducibility; -1 for random. |
| `webhook_url` | string | No | URL to receive a POST request when the video finishes generating. |

## Examples

### cURL
```bash
curl -X POST https://api.kenerateai.com/api/v3/kenerateai/wan-2.1/i2v-720p \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "image": "https://example.com/source-image.jpg",
    "prompt": "The camera pans slowly across the landscape, clouds moving dynamically in the sky",
    "duration": "5",
    "num_inference_steps": 30
  }'
```

### Python
```python
import requests

url = "https://api.kenerateai.com/api/v3/kenerateai/wan-2.1/i2v-720p"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}
data = {
    "image": "https://example.com/source-image.jpg",
    "prompt": "The camera pans slowly across the landscape, clouds moving dynamically in the sky",
    "duration": "5"
}

response = requests.post(url, headers=headers, json=data)
print(response.json())
```

### Response Format
When the request is successfully accepted, the API responds with a `queued` status and an `id`:
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "id": "task_abc123",
    "status": "queued",
    "urls": {
      "get": "https://api.kenerateai.com/api/v1/predictions/task_abc123"
    }
  }
}
```

You can then poll the prediction endpoint or wait for a webhook callback.

## Pro Tips for Best Quality
1. **Use high-resolution, well-lit source images** for optimal results.
2. **Be specific in your prompt** — describe camera movement, subject actions, and atmospheric details.
3. **Start with lower inference steps (20–25)** for quick previews, then increase for final renders.
4. **Use negative prompts** to reduce artifacts like blur, distortion, or unwanted motion.
5. **Fix the seed** when iterating to compare the effect of different parameters.
