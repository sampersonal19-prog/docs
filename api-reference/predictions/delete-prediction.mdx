---
title: "Delete a Prediction"
api: "DELETE https://api.kenerateai.com/api/v2/predictions/{task_id}"
description: "Permanently delete a generation task and its associated files."
---

Use this endpoint to permanently delete a prediction record from your account. 

If the generation task completed successfully and produced an image hosted on the KenerateAI CDN, **this endpoint will also permanently delete the image files from our storage buckets**. This is highly recommended for privacy-conscious applications handling sensitive user generations.

### Path Parameters

<ParamField path="task_id" type="string" required>
  The unique ID of the generation task you want to delete.
</ParamField>

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer authentication header. Example: `Bearer YOUR_API_KEY`
</ParamField>

### Request Example

```bash cURL
curl --request DELETE \
  --url https://api.kenerateai.com/api/v2/predictions/task_1234567890 \
  --header 'Authorization: Bearer YOUR_API_KEY'
```

### Response

<ResponseField name="ok" type="boolean">
  `true` if the prediction was found and successfully deleted.
</ResponseField>

<ResponseField name="message" type="string">
  A descriptive message confirming the deletion.
</ResponseField>

```json Example Response
{
  "ok": true,
  "message": "Prediction deleted"
}
```

<Note>
  If the `task_id` does not exist or has already been deleted, the API will return a `404 Not Found` error.
</Note>
