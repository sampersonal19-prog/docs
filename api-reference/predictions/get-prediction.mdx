---
title: "Get a Prediction"
api: "GET https://api.kenerateai.com/api/v2/predictions/{task_id}"
description: "Retrieve the current status and output of a generation task."
---

If you are not using webhooks, you can manually poll this endpoint to check the status of your task. We recommend polling every 1 to 2 seconds.

### Path Parameters

<ParamField path="task_id" type="string" required>
  The unique ID returned when you first initiated the generation request.
</ParamField>

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer authentication header. Example: `Bearer YOUR_API_KEY`
</ParamField>

### Request Example

```bash cURL
curl --request GET \
  --url https://api.kenerateai.com/api/v2/predictions/task_1234567890 \
  --header 'Authorization: Bearer YOUR_API_KEY'
```

### Response

The response object contains a standard wrapper with the task's state inside the `data` field.

<ResponseField name="code" type="integer">
  HTTP status code (e.g., 200 for success)
</ResponseField>

<ResponseField name="message" type="string">
  Status message (e.g., "success")
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="properties" defaultOpen>
    <ResponseField name="id" type="string">
      The unique identifier for the task.
    </ResponseField>
    
    <ResponseField name="status" type="string">
      The current status of the task. Will be one of:
      * `queued`: Waiting to be picked up by a worker.
      * `processing`: The model is actively generating the image.
      * `completed`: The image has been generated successfully.
      * `failed`: The generation failed.
    </ResponseField>
    
    <ResponseField name="model" type="string">
      The model slug used for the generation (e.g. `kenerate-image`).
    </ResponseField>
    
    <ResponseField name="outputs" type="array">
      An array of strings containing the generated public image URLs or Base64 data URIs. Empty if the status is not completed.
    </ResponseField>

    <ResponseField name="urls" type="object">
      <Expandable title="properties">
        <ResponseField name="get" type="string">
          URL to retrieve the prediction result via polling
        </ResponseField>
      </Expandable>
    </ResponseField>
    
    <ResponseField name="error" type="string">
      The error message detailing why the task failed. Empty if no error occurred.
    </ResponseField>
    
    <ResponseField name="created_at" type="string">
      ISO timestamp of when the request was created.
    </ResponseField>

    <ResponseField name="timings" type="object">
      <Expandable title="properties">
        <ResponseField name="inference" type="integer">
          Inference time in milliseconds
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>
